The Akan Name Generator

This is a web application that is supposed to find out the day of the week you were born as well as assigning a ghananian name that depends on the day you were born. This information can be acquired using your date of birth as well as your sex.

For males the following are the names:

  • Sunday : Kwasi
  • Monday : Kwado
  • Tuesday : Kwabena
  • Wednesday : Kwaku
  • Thursday : Yaw
  • Friday : Kofi
  • Saturday: Kwame

For females the following are the names:

  • Sunday : Akuosa
  • Monday : Adwoa
  • Tuesday : Abenaa
  • Wednesday : Akua
  • Thursday : Yaa
  • Friday : Afua
  • Saturday : Ama
